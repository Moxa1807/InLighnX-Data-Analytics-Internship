## 📊 Project Highlights

### 01. E-Commerce Sales & Profitability Dashboard
An interactive Power BI dashboard developed to analyze e-commerce sales, profitability, product categories, sub-categories, states, and payment modes.

### 02. Shopify Sales Dashboard
A Power BI dashboard focused on analyzing Shopify sales performance, product types, payment gateways, revenue, orders, and short-term sales trends.

### 03. New Year Sales – Exploratory Data Analysis
A Python-based EDA project analyzing customer demographics, purchasing behavior, product categories, occupations, and state-wise sales patterns.

### 04. Zomato Restaurant Data – Exploratory Data Analysis
A Python EDA project exploring restaurant ratings, online ordering, table booking, restaurant categories, customer votes, and approximate costs.

### 05. Music Store Database Analysis
A SQL-based analysis of a relational music store database using joins, aggregate functions, subqueries, CTEs, and window functions to answer business questions.

### 06. Amazon Sales Dashboard
An interactive Excel dashboard developed to analyze sales performance across regions, states, cities, and product categories using Pivot Tables, formulas, charts, slicers, and KPIs.
